This folder contains data from dummy runs of the MB3 Marcus replication. 

Explanation of files by extensions: 
.edat = E-Prime 2.0 output of full runs, or aborted runs that have been recovered in E-Recovery
...-1.txt = E-Prime 2.0 output of full or aborted runs (format is not delimited in columns)
...-1-dummy.txt = files converted from .edat files to delimited columns
